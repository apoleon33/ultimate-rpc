# ultimate-rpc

A small script that add compatibility with discord rich presence for a lot of application.

# how to make it work?

```
git clone https://github.com/apoleon33/ultimate-rpc
cd ultimate-rpc
pip install -r requirements.txt
python3 main.py
```
